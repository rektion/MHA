This is a Robotic arm using Ultrasounds to levitate objects. This technologies is based on acoustic levitation, a principle thats has been well known for over 80 years. So with this we can move and take small objects without touching them. It could be usefull in some inductries to not alter any properties of an object.
This is still a prototype in his early phases.
