There is an energy loss because the battery carry by the robot need to charge and then it is one more step for electricity.
Volkswagen unveiled its new concept. It is a mobil charging point for electric vehicles. Actually it is a robot that can carry charging modules and plug it on electric vehicles. All you have to do is activate the robot by an application and it does everything, you can stay in the car. Therefore there is no need for complexe infrastructure because cars can park anywhere near the station and robots move toward cars.
